Debug to Learn

An AI-powered learning platform that helps students develop debugging skills through interactive coding assignments.

The Problem: As AI-generated code becomes more common, students need opportunities to practice debugging real programming mistakes. Debug to Learn gives instructors an easy way to generate debugging exercises while helping students build one of the most valuable programming skills.

What I Built: I developed the backend database architecture that stores student accounts, instructor information, assignments, and AI prompts. I also implemented the database integration with FastAPI so professors can create assignments using an AI chatbot that automatically generates debugging activities.

Tech Stack: Python, FastAPI, PostgreSQL, SQL, OKD, Git, HTML, CSS, JavaScript

Key Features:

  • AI chatbot assists professors in creating debugging assignments
  • Students submit completed work and receive AI-generated feedback
  • Instructors can edit assignment content, deadlines, and settings
  • Secure database stores users, assignments, and generated prompts

What I Learned: I strengthened my SQL database development skills by integrating PostgreSQL with FastAPI, while also gaining experience using GitHub Projects to coordinate work across a large software engineering team.

NASA Space Explorer

An interactive web app for exploring NASA's Astronomy Picture of the Day archive.

The Problem: NASA provides incredible astronomy data through public APIs, but browsing that information isn't always user-friendly. This project creates an intuitive interface for discovering space imagery and learning about astronomical events.

What I Built: I built a responsive front-end application that connects to the NASA API and displays Astronomy Picture of the Day entries. Users can browse images, search by date, and read detailed descriptions of each featured event.

Tech Stack: HTML, CSS, JavaScript, NASA API

Key Features:

  • NASA API integration
  • Date-based search
  • Responsive interface
  • Detailed astronomy information

What I Learned: I gained experience consuming REST APIs with JavaScript while improving my responsive web design skills.

Scooby-Doo Statistical Analysis

A statistical research project analyzing trends in Scooby-Doo episodes using real-world data.

The Problem: Television datasets provide an engaging way to apply statistical methods to real data. This project explored whether episode characteristics were associated with viewer ratings and other external factors.

What I Built: I analyzed the Scooby-Doo dataset in R using data cleaning, visualization, regression analysis, and hypothesis testing. The final project communicates statistical findings through a research paper and supporting visualizations.

Tech Stack: R, ggplot2, dplyr, tidyverse

Key Features:

  • Data cleaning and preprocessing
  • Statistical modeling
  • Data visualization
  • Research paper summarizing findings

What I Learned: I improved my statistical modeling skills while gaining experience presenting technical analyses in a professional research format.
